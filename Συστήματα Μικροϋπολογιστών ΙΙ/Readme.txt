Οι κώδικες έχουν χωριστεί σε 4 διαφορετικά αρχεία.
\begin{itemize}
\item a3a.asm .Περιέχει την υλοποίηση για τον υπολογισμό συχνότητας με χρονικό παράθυρο 1sec.
\item a3b.asm Περιέχει την υλοποίηση για τον υπολογισμό συχνότητας με Input Capture και τη διαφορά χρόνου 4 παλμών.
\item b1.asm Περιέχει την υλοπίηση για τον έλεγχο κινητήρα με pwm, πατώντας κουμπιά.
\item b2.asn Περιέχει την μη επαληθευμένη υλοποίηση για τον αυτόματο έλεγχο κινητληρα με pwm.
